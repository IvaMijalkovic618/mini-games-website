# Mini Games Web App

This project is a simple web application that allows users to enter a username and access a collection of interactive browser-based games.

After successful username validation, users are redirected to a main menu where they can choose and play different games.

## Live Demo
https://ivamijalkovic618.github.io/mini-games-website/

## Features
- Username validation (input checking and restrictions)
- Navigation between multiple pages
- Interactive game menu
- Memory Game with matching logic
- Treasure Hunt game with grid-based interaction
- Logout functionality

## Technologies
- HTML
- CSS
- JavaScript
